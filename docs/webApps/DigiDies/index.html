What the app does

It's a simple pwa to roll dice of different sizes

Features include

  • Create dice from presets
  • Create dice from input
  • Roll one or multiple dice
  • See a list of past rolls to prevent cheating
  • Add an icon to your dice

Missing: A11Y check, deleting / undo a single dice, stay tuned for future updates.

PWA intro

DigiDies was built to learn more about PWAs.

A PWA is a progressive web app - basically a programm running in the Browser. A PWA can be installed from different browsers and used offline.

Service workers ensure that it works offline, caching things. If done correctly, they should automatically update things as soon as there is web access

Uninstall it like any web app
