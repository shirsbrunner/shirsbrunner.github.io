Living in the moment

Publication: 26 March, 2026

This feels like a small thing to do, but maybe it isn't. I read a lot of people having problems updating now pages. Maybe you just don't feel like sharing some things with the world? Anyway, I'll give it a try and see what happens.

Whut? – I'm glad you ask. This is a now page, and if you have your own site, maybe you should make one, too?
